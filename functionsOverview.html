Traditionally Invoked Functions

  • Invoking a function stops the execution
  • Ex. of Traditional invocation ---> plus (a,b);
  • With traditional invocation, "this" ouputs as the global object
  • By looking at the console, we can see that the the window object has been outputed
  • In this example, we are returning values as parameters. return ()
function plus(a,b) { return ( console.log(a+b), console.log(this), console.log(arguments) ) } plus(5,2);

Using Functions as Objects (Invoking as Methods)

  • A method is a function that has been assigned as a property of an object.
  • As you can see on the console, "this" now returns the object where its scope currently resides. Which is calc
var calc = { theStatus:"awesome", plus: function (a,b) { return ( console.log(this), console.log(a + b), console.log(arguments), console.log(this.theStatus) ) } } calc.plus(5,6);

Invoking Functions as Constrictors

  • Functions can constuct objects
  • We use the "new" keyword and this method is called constuctor invocation
  • "new" creates a new instance of the dog object with breed: "Doverman", name: "Rover"
  • "this" points to the instance of the object
var Dog = function () { var name, breed; return console.dir(this); // returns directory of object } firstDog = new Dog; firstDog.name = "Rover"; firstDog.breed = "Doverman"; console.log(firstDog.name); secondDog = new Dog; secondDog.name = "Fluffy"; secondDog.breed = "Poodle"; console.log(secondDog.name);

Expanding Objects through Prototype

  • Javascript is known as a prototypical inheritance language
  • You can give the functionality of an object to another object
  • prototype object gives you access
  • So by looking at Cat.prototype.Catspeak = speak; We are accessing the prototype of Cat And giving it a method called Catspeak That will inherit the functionality from the function speak
  • All Objects inherit properties
function speak (sayWhat) { console.log(sayWhat); } var Dog = function () { var name, breed } var Cat = function () { var name, breed; } Cat.prototype.Catspeak = speak; //Gives the cat the ability to speak cat Dog.prototype.Dogspeak = speak; //Gives the cat the ability to speak dog firstCat = new Cat; firstCat.name = "Chococat"; firstCat.breed = "Bengal"; firstCat.Catspeak("Meow"); console.log(firstCat); firstDog = new Dog; firstDog.name = "Rover"; firstDog.breed = "Doberman"; firstDog.Dogspeak("Woof"); // console.log(firstDog);
